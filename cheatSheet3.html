Html for Lists

  1. Start tags for
    • an ordered list
      • - <ol>
    • an unordered list
      • - <ul>
    • a description list
      • - <dl>
    • a list item
      • - <li>
    • a description term
      • - <dt>
    • a description data
      • - <dd>

Css for Lists:

  1. Make bullet a square
    2. ul.a { list-style-type: square;}
  2. Make bullet an image named "my-bullet.png"
    3. ul { list-style-image: url('my-bullet.png'); }
  3. Make bullet an uppercase roman number
    4. ol.b { list-style-type: upper-roman; }
  4. Set bullet position to be inside the list item's text flow
    5. ul.b { list-style-position: inside; }
  5. Set bullet position to be outside the list item's text flow
    6. ul.d { list-style-position: outside; }

Html for Tables

  1. Start tags for
    • a table
      • <table>
    • a tabel row
      • <tr>
    • a table header
      • <thead>
    • a tabel header cell
      • <th>
    • a table body
      • <tbody>
    • a table cell
      • <td>
    • a table footer
      • <tfoot>
    • a table footer cell
      • <td>

Css for Tables

  1. Black dotted border of 1 px width
    2. table { border: 1px dotted black; }
  2. Solid border of 1 px width having a red color and rounded corners
    3. table { border: 1px solid black; border-color: red; border-radius: 5px; }
  3. Merge the borders of table cells so that they appear as one line
    4. table { border-collapse: collapse; }
